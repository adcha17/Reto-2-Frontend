Click, DblClick

click() : Define un evento al momento de hacer click.

Sintaxis: $('selector').click(function(){});

dblclick() : Define evento al momento de hacer doble click.

Sintaxis: $('selector').dblclick(function(){});




MouseLeave, MouseEnter, Hover

mouseleave() : Define evento al momento en que el puntero del mouse deja la ubicacion del elemento.

Sintaxis: $('selector').mouseleave(function(){});

mouseenter() : Define evento al momento en que el puntero del mouse ingresa a la ubicacion del elemento.

Sintaxis: $('selector').mouseenter(function(){});

hover() : Permite definir los eventos mouseenter y mouseleave en una sola funcion.

Sintaxis: $('selector').hover(function(){});



Focus, Blur

focus() : El evento focus se activa cuando se ingresa con el puntero del mouse a un campo del formulario.

Sintaxis: $('selector').focus(function(){});

blur() : El evento blur se activa cuando se deja el campo.

Sintaxis: $('selector').blur(function(){});

FadeIn, FadeOut, FadeToogle, FadeTo

fadeIn() : Método para descolorar un elemento.

Sintaxis: $('selector').fadeIn(speed);

fadeOut() : Método para colorear un elemento.

Sintaxis: $('selector').fadeOut(speed);

fadeToggle() : Alterna entre los métodos fadeIn() y fadeOut().

Sintaxis: $('selector').fadeToogle(speed);

fadeTo() : Método para controlar la opacidad de un elemento siguiendo un parametro de referencia.

Sintaxis: $('selector').fadeTo(speed, opacity);
